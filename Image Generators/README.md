# Image Generators

R scripts that generate the figures embedded in worksheets and notebooks in this repo.

Each script is self-contained: it loads `library(coursekata)`, sources any beta
functions it needs from [coursekata/beta-functions](https://github.com/coursekata/beta-functions),
and writes its PNGs to the current working directory.

| Script | Generates figures for |
|---|---|
| `gen_img_7.9-7.10_visualizing_pre.R` | 7.9-7.10 Visualizing PRE worksheet (Chapter 07) and its companion notebook |
